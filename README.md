# SDE Sheet Tracker

A sleek, minimalist, and high-contrast React application designed to help developers track their progress through the 191-problem SDE Sheet. Built as a fast, modern single-page application, it provides an uninterrupted and visually pleasing experience.

## ✨ Features

- **Strictly Monochrome UI:** Designed with a zero-distraction philosophy. Pure blacks, crisp whites, and smooth grays.
- **Dynamic Progress Tracking:** A sticky, frosted-glass header dynamically updates a satisfying progress bar as you crush problems.
- **Local Persistence:** Your progress is automatically saved to the browser's `localStorage`. Pick up exactly where you left off, even after refreshing or closing the tab.
- **Organized Structure:** 191 curated problems elegantly structured across 27 distinct "Days" or topics ranging from Arrays to Dynamic Programming and Tries.
- **Smooth Interactions:** Custom-styled, high-contrast checkboxes and strikethrough animations make checking off tasks feel incredibly rewarding.

## 🚀 Tech Stack

- **React 18** (Functional Components, Hooks)
- **Vite** (Next Generation Frontend Tooling)
- **Vanilla CSS** (CSS Variables, Flexbox, CSS Transitions)

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/sde-sheet-tracker.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd sde-sheet-tracker
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in browser:**
   Navigate to `http://localhost:5173/` and start tracking your progress!
